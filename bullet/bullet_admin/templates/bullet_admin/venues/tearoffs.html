This interface allows you to generate printable tear-off for your venue. A tear-off is the document with problems and barcodes for teams.

The problems on a tear-off can be ordered in two ways:

  • Aligned: Problems are sorted so that when you print the whole document, problems from the same team will be under each other, so you can just cut the papers and have them already ordered. For example, you will get one page with the first problem for the first four teams, then another page with the second problem and so on.
  • Sequential: Problems are sorted sequentially on the page, meaning you will need to order them manually. For example, you will get a page containing problems 1–4 for the first team, then another page containing problems 5–8, and so on.
